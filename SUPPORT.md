# 支持说明

感谢你使用或关注 `考研专注`。为了让维护者能更快处理问题，请按类型选择合适入口。

## 使用问题

如果你不确定某个功能如何使用，请先阅读：

- [README.md](./README.md)
- [FEATURES.md](./FEATURES.md)
- [项目完整链路说明.md](./项目完整链路说明.md)

仍然无法解决时，可以提交 GitHub issue，并尽量说明应用版本、Windows 版本、你已经尝试过的步骤。

## Bug 报告

请使用 Bug report 模板，提供：

- 应用版本和 Windows 版本
- 是否在 Tauri 桌面端运行
- 是否启用 WebDAV、对象存储、飞书或邮件提醒
- 最小复现步骤
- 期望行为和实际行为
- 已脱敏的截图或日志片段

不要上传真实数据库、同步备份、完整日志、密钥、token、邮箱授权码或私人路径。

## 功能建议

请使用 Feature request 模板，并说明：

- 真实学习或维护场景
- 期望能力
- 可接受的替代方案
- 是否愿意协助测试或提交 PR

## 安全问题

安全问题不要直接公开细节。请阅读 [SECURITY.md](./SECURITY.md)，优先使用 GitHub Private Vulnerability Reporting / Security Advisory；如果该功能暂不可用，请先提交不含敏感信息的 issue 请求私下沟通。

## 维护范围

当前公开维护重点是 Windows / Tauri 桌面端。Android 相关内容不属于当前公开仓库主维护范围。
